## Welcome to DataS-DHSC
**Please note that we are currently in the progress of migrating to our new GitHub organisation [dhsc-govuk-internal](https://github.com/dhsc-govuk-internal) and this organisation will eventually be archived as read-only on 28 September 2026.**

If you have not already received an invitation to the new organisation, or you have any other issues, please contact the organisation owners via GitHub [AT] dhsc [DOT] gov [DOT] uk.

We strongly recommend that any new project(s) are started in the new organisation and that you begin planning how best to migrate any existing active project(s) across. Details on how to set up a new project or migrate an existing one can be found in our new GitHub userguide (you will need to be a member of the new organisation to access this guide).

## Setup and Access
For guidance on setting up Git and GitHub, including how to get access to the the new official GitHub organisation [dhsc-govuk-internal](https://github.com/dhsc-govuk-internal/), please see:  
Data Science Front Door > Data Platforms and Software > Software Installation and Tooling > Git Installation Guide

## Key migration dates

**30 June 2026**  
All members will be removed from this organisation.
Users will retain access to any repositories they currently have access to, but as an outside collaborator.
Users will no longer be able to create new repositories in the organisation.

**28 September 2026**  
All repositories in the organisation will be archived (read-only).  
Nothing will be deleted as part of this process. However, access to repositories will be removed if they are not migrated before archiving.  
**Please ensure any repositories you wish to keep are migrated before this date.**
